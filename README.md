#  Digital Wellbeing Analysis Dashboard (Power BI)

## Project Overview
This project analyzes smartphone usage patterns and digital wellbeing trends using Power BI.  
It provides insights into screen time, app usage, battery consumption, and user behavior segmentation.

---

## Tools & Technologies
- Power BI Desktop
- Power Query
- DAX
- Data Modeling

---

##  Dataset
- Source: Simulated user behavior dataset
- Features include:
  - Screen Time
  - App Usage
  - Battery Drain
  - Data Consumption
  - Age, Gender, Device Model

---

##  Key Features
- Interactive dashboard with KPI cards
- Slicers for:
  - Age Group
  - Gender
  - Device Model
  - Behavior Category
- Visual insights using bar charts, donut charts, and cards

---

##  Key Insights
- High screen time observed among specific age groups
- Strong correlation between app usage and battery drain
- Behavior segmentation highlights heavy vs moderate users
